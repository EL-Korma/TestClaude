# SKILL: Pricing Strategy
> TwinFit — Free tier, premium plans, price testing

## Tier Design
```
FREE:
  Gym streak tracking
  Daily pose logging
  Bronze evolution
  Basic history (7 days)
  3 AI recipes/day
  3 meal scans/day
  Ads displayed

PREMIUM ($4.99/mo or $34.99/yr):
  Everything in Free
  Ad-free
  2 streak freezes/month
  Silver + Gold + Platinum evolution
  Unlimited history
  Unlimited AI recipes
  Unlimited meal scans
  Early access to new features
```

## Price Positioning
```
Competitor benchmarks:
  MyFitnessPal Premium: $19.99/mo
  Strava Summit:        $11.99/mo

TwinFit monthly: $4.99 = "cheapest premium fitness app"
TwinFit annual: $34.99 = "$2.92/mo — less than one coffee"

Anchor on annual (higher LTV, lower churn)
Show annual as "MOST POPULAR" in paywall
```

## A/B Test Plan
```
Test 1: $4.99 vs $5.99/mo (elasticity check)
Test 2: $29.99 vs $34.99 vs $39.99/yr
Test 3: 7-day free trial vs no trial
Test 4: Annual-first vs monthly-first in paywall layout

Measure LTV, not just conversion rate
Minimum: 2 weeks per test, 200+ paywall views
```

## Streak Freeze as Monetization Hook
```
Free users:    0 freezes (see the option but can't use it)
Premium users: 2 freezes/month

When free user taps freeze → paywall opens
This is highest-converting paywall entry point
Feature is most emotionally resonant (prevent losing progress)
```
