What is TwinFit?

A paired fitness app where two people keep each other accountable. Both must check in daily to maintain their shared streak. Core loop: photo check-in → streak → quests → dumbbell economy → shop rewards.

🔥
Duo Streaks
Both partners log the same day or streak resets. Personal + group streaks tracked separately in DB.
📸
Photo Check-ins
Camera uploads to Supabase Storage CDN. Public URL stored in DB so photos persist across devices.
🧬
AI Nutrition
Voice or text → Claude Haiku returns calories, protein, carbs, fat, healthScore + generates recipes.
🏋️
Dumbbell Economy
Earn via quests. Spend on XP boosts (2–3×), 15 avatar border tiers, and packs.
⚡
Evolution Tiers
Bronze→Silver→Gold→Platinum→Legendary. Animated glow nodes on EvolveScreen.
🔔
Push Notifications
Expo tokens saved to backend on login. Streak reminders at 8PM daily. Android channels configured.

Auth Flow

JWT-based, stateless backend. Token lives in expo-secure-store.

backend/lib/auth.tsTS
export function getUserId(req) {
  const token = req.headers
    .get("authorization")
    ?.replace("Bearer ", "");
  try {
    const { sub } = jwt.verify(token, secret);
    return sub; // userId
  } catch { return null; }
}
export function signToken(userId) {
  return jwt.sign(
    { sub: userId }, secret, { expiresIn: "7d" }
  );
}
Every protected routeTS
export async function GET(req) {
  const userId = getUserId(req);
  if (!userId) return unauthorized(); // 401
  const data = await prisma.model
    .findMany({ where: { userId } });
  return Response.json({ data });
}
AuthStore — applyAuth()TS
const applyAuth = async (res) => {
  setToken(res.token);        // in-memory
  await storeToken(res.token); // SecureStore
  await fetchAndSetUser(res.token);
  registerPushToken().catch(() => {});
};

Why Lazy Prisma Init?

Problem: Next.js 15 executes route modules at build time. new PrismaClient() at module level crashes the build.

Fix: lib/prisma.ts uses a Proxy — PrismaClient is only created on first actual DB property access (at request time, not build time).
backend/lib/prisma.tsTS
export const prisma = new Proxy({}, {
  get(_, prop) {
    return getClient()[prop]; // lazy
  },
});
